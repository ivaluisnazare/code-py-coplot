# Resolvendo código Python com o Github Copilot
Aqui, veremos algumas soluções de código em Python usando o Github Copilot.

## 1 - Concatenando dados
O código concatenating_data.py recebe dois dados diferentes do usuário e os concatena em uma única string.

## 2 - Repetindo Textos
O código repeating_texts.py recebe um texto e um número do usuário e repete o texto o número especificado de vezes.

## 3 - Operações Matemáticas Simples
O código simple_operations.py recebe dois números e uma operação matemática (adição, subtração, multiplicação ou divisão) e retorna o resultado da operação.

## 4 - Verificando Números Pares e Ímpares
O código checking_even_and_odd_numbers.py verifica se um número fornecido pelo usuário é par ou ímpar.

## 5 - Calculando Média de Notas
O código calculation_grade_average.py calcula a média de três notas fornecidas pelo usuário.

## 6 - Verificando Palíndromos
O código checking_palindromes.py verifica se uma palavra fornecida pelo usuário é um palíndromo (lê-se da mesma forma de trás para frente).
